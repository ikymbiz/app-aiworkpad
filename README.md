# app-aiworkpad

## 概要・説明
**AIWorkPad** は、ブラウザ上で軽快に動作し、AIアシスタント機能を統合した次世代のモダンMarkdownエディタです。

Google Gemini APIを活用し、文章の推敲、要約、構造化、そして深掘り調査までをシームレスにサポートします。データはすべてブラウザのローカルデータベース（IndexedDB）に保存されるため、アカウント登録不要でプライバシーを保ちながら安全かつ高速に利用できます。

PC・モバイルの両方に最適化されたレスポンシブデザインを採用しており、いつでもどこでも快適な執筆活動を強力にバックアップします。

## 主な機能一覧

### ✍️ 洗練されたMarkdownエディタ
- **リアルタイム保存機能**: 5分ごとの定期自動保存に加え、入力停止時のバックグラウンド保存を搭載。
- **入力支援ツールバー**: 見出し、太字、リスト、引用などのMarkdown記法をワンクリックで挿入可能。
- **プレビュー機能**: 執筆中のテキストをワンタッチでWYSIWYGライクなリッチテキスト表示に切り替え。

### ✨ テキスト選択連動型AIアクション（フローティングメニュー）
エディタ内のテキストを選択するだけで、以下のAI処理を瞬時に実行できます。
- **きれい化**: 誤字脱字の修正や冗長な表現の整理。
- **要約**: 選択した長文の要点を抽出。
- **表現修整**: 文章をより専門的で魅力的なトーンへブラッシュアップ。
- **構造化**: ベタ打ちのテキストを適切なMarkdown（見出し、箇条書きなど）に整理。
- **深掘調査**: 選択テキストを元にAIが詳細な調査を行い、レポートを作成して保存。

### 💬 文脈を理解するAIチャット
- ドキュメント全体や選択中のテキストをコンテキスト（文脈）としてAIに送信可能。
- 質問に対するAIの回答を、ワンクリックで本文へ直接挿入できます。

### 📂 高度なドキュメント管理・履歴
- **複数ドキュメント管理**: サイドバーから複数のノートを作成・切り替え・削除が可能。
- **自動目次生成（TOC）**: 本文中の見出しタグ（`#`）を自動検出し、クリック可能な目次を生成。
- **バージョン履歴**: AIによる修正、自動保存、手動保存のタイミングで履歴を記録。いつでも過去のバージョンを復元できます。
- **調査ノート**: 深掘り調査の結果を個別のタブにストックし、後から本文に組み込むことが可能。

## 使用技術・ライブラリ
本プロジェクトは、ビルドツール不要の構成で作成されており、以下の技術スタックとCDNライブラリを使用しています。

- **コア言語**: HTML5, CSS3, Vanilla JavaScript
- **スタイリング**: Tailwind CSS (CDN版 / Typographyプラグイン使用)
- **アイコン**: Font Awesome (v6.4.0)
- **Markdown解析**: Marked.js
- **ローカルデータベース**: localForage (IndexedDBラッパー)
- **AI連携**: Google Gemini API (`gemini-2.5-flash` モデル)

## セットアップ・使い方

### 1. 起動方法
サーバーの構築や依存パッケージのインストールは不要です。
ダウンロードした `index.html` ファイルを、Google Chrome, Safari, Edge などのモダンブラウザで直接開くだけで起動します。

### 2. 初期設定（APIキーの登録）
AI機能を利用するために、Google Gemini APIキーの設定が必要です。
1. 画面右上の **「設定アイコン（歯車）」** をクリックします。
2. 取得した **Gemini APIキー** を入力し、「保存」をクリックします。
   *(※ APIキーはブラウザのローカルストレージにのみ安全に保存されます)*

### 3. 基本的な使い方
- **執筆・編集**:
  左サイドバー（モバイルの場合は左上のフォルダアイコン）から「＋」ボタンを押して新規ドキュメントを作成します。中央のエディタエリアで自由にMarkdownを入力してください。
- **AI機能の呼び出し**:
  エディタ上のテキストをドラッグして選択すると、AIメニュー（アクションシート）がポップアップします。実行したいアクション（要約、きれい化など）を選択してください。
- **サイドパネルの活用**:
  画面右側（モバイルでは右上のツールアイコン）のパネルから「チャット」「目次」「調査」「履歴」の各タブにアクセスできます。チャット機能では、下部のチェックボックスをオンにすることで、執筆中の文章をAIに読み込ませた上で対話が可能です。
