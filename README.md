# AIWorkPad

## 概要・説明
**AIWorkPad** は、ブラウザ上で完結するAI搭載の高機能Markdownエディタです。
Google Gemini および Anthropic Claude API と連携し、日々のドキュメント作成や文章の推敲を強力にサポートします。

選択したテキストに対する即時AIアクション（要約、詳細化、構造化など）や、文脈を理解したAIチャット、ゼロからの記事ドラフト生成機能などをシームレスに統合しています。データはすべてブラウザのローカルデータベース（IndexedDB）に保存されるため、サーバーの構築やバックエンドは一切不要で、HTMLファイルをブラウザで開くだけですぐに利用を開始できます。スマートフォンとPCの両方に最適化されたレスポンシブデザインを採用しています。

## 主な機能一覧

### 📝 高機能マークダウンエディタ
- 入力支援ツールバー（見出し、太字、リスト、引用、コード、リンク）
- リアルタイムプレビュー切り替え機能
- 見出し（#）から自動生成される目次（TOC）ナビゲーション
- 定期自動保存およびバージョン履歴管理（過去のバージョンへの復元対応）
- 複数ドキュメントのローカル管理

### 🤖 選択テキストへのAIアクション（コンテキストメニュー）
エディタ内のテキストを選択するとポップアップメニューが表示され、以下の処理をワンクリックで実行し、元のテキストを置き換えます。
- **詳細化**: 前後の文脈を読み取り、内容を膨らませて加筆
- **きれい化**: 誤字脱字の修正、冗長な表現の整理
- **要約**: 長い文章を簡潔に要約
- **表現修整**: より専門的で魅力的な文章にブラッシュアップ
- **構造化**: 内容に応じてMarkdown形式で適切にリスト化・構造化
- **深掘り調査**: 選択したトピックに対してAIに詳細調査を指示し、結果を挿入

### 💬 コンテキスト対応AIチャット
- 作成中のドキュメント全体、または選択中のテキストを「前提知識」としてAIに共有しながら対話が可能。
- チャットの回答をワンクリックでエディタ内のカーソル位置に挿入。

### ✍️ 記事ドラフト（生成）機能
- テーマやターゲット層などの指示を入力するだけで、ゼロから記事のドラフト（下書き）を作成。
- 生成した記事は、現在のドキュメントの末尾に追加、または新規ドキュメントとして保存可能。

### ⚙️ 高度なカスタマイズとパフォーマンス
- **マルチモデル対応**: Gemini 2.5 Flash と Claude 3 Haiku の切り替えが可能。
- **AIパーソナリティ管理**: 「記事ライター」「要約専門家」など、独自のシステムプロンプト（役割）を自由に追加・切り替え。
- **Web Worker対応**: AIの通信処理をバックグラウンドで実行し、生成中もエディタのUIがフリーズしない設計。
- **スリープ防止（Wake Lock）**: AIの長文生成中にスマートフォンの画面が暗くなるのを防止。

## 使用技術・ライブラリ

本プロジェクトはビルドツールを必要とせず、ピュアなWeb技術とCDNライブラリのみで構成されています。

- **コア技術**: HTML5, CSS3, Vanilla JavaScript
- **スタイリング**: Tailwind CSS (CDN / Typographyプラグイン)
- **アイコン**: FontAwesome 6 (CDN)
- **マークダウン解析**: Marked.js
- **ローカルストレージ**: LocalForage (IndexedDBラッパー)
- **API通信**: Fetch API (Web Worker経由でGemini / Claude APIと通信)

## セットアップ・使い方

本アプリケーションはサーバー不要のSPA（シングルページアプリケーション）です。

### 1. 起動方法
1. リポジトリの `index.html`（または該当のHTMLファイル）をダウンロードします。
2. Google Chrome, Microsoft Edge, Safari などのモダンブラウザでファイルを開きます。（ダブルクリックするか、ブラウザにドラッグ＆ドロップしてください）

### 2. 初期設定（APIキーの登録）
AI機能を利用するためには、ご自身のAPIキーが必要です。
1. 画面右上の **歯車アイコン（設定）** をクリックします。
2. お持ちの **Gemini APIキー** または **Claude APIキー** を入力します。
   - ※ 無料枠のあるGemini APIの利用を推奨します。Google AI Studioから取得可能です。
3. デフォルトのAIモデルを選択し、「保存」をクリックします。
   - ※ APIキーはブラウザのローカルストレージに暗号化されずに保存されます。共有PCなどでの利用にはご注意ください。

### 3. 基本的な使い方
- **文章を書く**: メインエリアにMarkdown形式で文章を入力します。
- **AIアシストを使う**: エディタ内のテキストをドラッグして選択すると、「AIメニュー」が表示されます。用途に合わせてボタン（詳細化、要約など）をクリックしてください。
- **ツールパネルを開く**: PCでは画面右側、モバイルでは右上のスパナアイコンからサイドパネルを開き、チャット機能や記事生成機能にアクセスできます。
- **プレビューする**: ツールバー右端の「目のアイコン」をクリックすると、Markdownのプレビュー表示に切り替わります。
- **ドキュメント管理**: 左上のフォルダアイコン（またはPC版の左サイドバー）から、ドキュメントの新規作成や切り替えが行えます。自動保存機能が搭載されています。
