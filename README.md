# app-aiworkpad

## 概要・説明
**AIWorkPad**は、Google Gemini APIを活用し、ブラウザ上で完結するAI搭載型のMarkdownドキュメントエディタです。

バックエンドサーバーやデータベースの構築は一切不要で、1つのHTMLファイルをブラウザで開くだけですぐに利用できます。ユーザーの入力したデータや設定はすべてブラウザのローカル（IndexedDB / LocalStorage）に安全に保存されます。

文章の執筆をサポートするリッチなMarkdown入力補助に加え、テキストを選択するだけで即座に呼び出せる「フローティングAIメニュー」を搭載。文章の校正（きれい化）、要約、表現の修整、構造化、そして深掘り調査まで、執筆作業をAIがシームレスにアシストします。また、PC・スマートフォンの両方で快適に操作できるレスポンシブデザインを採用しています。

## 主な機能一覧

### 📝 ドキュメント編集 & 管理
* **Markdownエディタ＆プレビュー**：入力支援ツールバーによる直感的なMarkdown入力。ボタン一つでWYSIWYGライクなプレビュー画面へ切り替え可能。
* **リスト自動連番入力**：Enterキーを押すだけで、箇条書きや番号付きリスト（1., a., I. など）を自動で継続。
* **ローカル自動保存**：定期的な自動保存と手動保存に対応。データはすべてブラウザ内（IndexedDB）に保存されます。
* **ドキュメント管理**：複数のドキュメントを作成・切り替え・削除可能。左サイドバーで簡単に管理できます。
* **動的目次（TOC）**：本文の見出し（Hタグ）から自動で目次を生成し、クリックで該当箇所へスクロールします。

### ✨ AIアシスト機能
* **フローティングAIメニュー**：エディタ上でテキストを選択すると自動でアクションシートがポップアップし、以下の処理をワンクリックで実行できます。
  * **きれい化**：誤字脱字の修正、冗長な表現の削除
  * **要約**：文章を簡潔にサマライズ
  * **表現修整**：より専門的・魅力的な文章へブラッシュアップ
  * **構造化**：見出しや箇条書きを用いて論理的なMarkdownへ変換
  * **深掘調査**：選択したテキストについてAIが詳細な調査レポートを作成
* **AIチャットアシスタント**：右サイドパネルに専用のチャットを搭載。ドキュメント全体や選択中のテキストを「コンテキスト（文脈）」として含めた上でAIに質問や指示を出すことができ、回答をワンクリックで本文に挿入できます。
* **調査ノート**：深掘り調査の結果は専用タブにストックされ、いつでも振り返りや本文への挿入が可能です。

### ⏳ 履歴・バックアップ
* **バージョン履歴管理**：自動保存、手動保存、AIによるテキスト書き換え前の状態を自動的に履歴として保存。いつでも過去のバージョンを復元できます。

## 使用技術・ライブラリ

* **フロントエンド**
  * HTML5 / CSS3 / Vanilla JavaScript (シングルページアプリケーション構成)
  * [Tailwind CSS](https://tailwindcss.com/) (UIスタイリング・レスポンシブ対応) + Typography Plugin
  * [FontAwesome](https://fontawesome.com/) (UIアイコン)
* **テキスト処理・データ管理**
  * [Marked.js](https://marked.js.org/) (Markdownの高速なパースとHTML変換)
  * [localForage](https://localforage.github.io/localForage/) (IndexedDBの非同期ラッパー、大容量のローカルデータ保存)
* **AI API**
  * [Google Gemini API](https://aistudio.google.com/) (`gemini-2.5-flash` モデルを使用)

## セットアップ・使い方

### 1. 起動方法
サーバー等の環境構築は不要です。ソースコードである `index.html`（または任意のファイル名）をダウンロードし、Google Chrome、Edge、Safariなどのモダンブラウザで開くだけでアプリケーションが起動します。

### 2. 初期設定（APIキーの登録）
AI機能を利用するために、Google GeminiのAPIキーを設定します。
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、無料のGemini APIキーを取得します。
2. アプリケーション画面右上の **歯車（設定）アイコン** をクリックします。
3. 取得したAPIキーを入力し、「保存」をクリックします。
   *(※ APIキーはブラウザのLocalStorageにのみ保存され、外部サーバーには一切送信されません。)*

### 3. 基本的な使い方
* **執筆する**: 中央のエディタ領域に文章を入力します。上部のツールバーを使ってMarkdownの書式を簡単に適用できます。
* **AIメニューを呼び出す**: エディタ上で任意のテキストをドラッグして選択すると、AIメニュー（きれい化、要約など）がポップアップします。
* **チャット・調査・履歴**: 画面右上のツールアイコン（モバイルの場合）または右サイドパネルから、AIとのチャット、目次の確認、調査ノートの閲覧、バージョン履歴の復元が行えます。
* **プレビュー**: ツールバー右側の「プレビュー」ボタンを押すと、Markdownがレンダリングされた読みやすい表示に切り替わります。プレビュー中も目次からのジャンプが可能です。
