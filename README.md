# app-aiworkpad

## 概要・説明
AIWorkPadは、AIを活用して文章作成を強力にサポートする、ブラウザ完結型のマークダウンテキストエディタです。
エディタ上のテキストを選択するだけでAIメニューがポップアップし、文章の校正、要約、構造化、さらには深掘り調査を直感的に実行できます。

また、IndexedDBを利用したローカルファーストな設計になっており、入力中のリアルタイム自動保存や、詳細なバージョン履歴（バックアップ）機能を備えています。PCブラウザだけでなく、スマートフォン向けに最適化されたアクションシートやドロワーUIを採用しており、モバイル環境でも快適に執筆作業を行うことができます。

## 主な機能一覧

### 🤖 コンテキストAIアシスタント
エディタ内のテキストを選択すると即座にAIメニューが表示され、以下の処理を実行できます。
- **きれい化**: 誤字脱字の修正や冗長な表現の削除
- **要約**: 長い文章を要点だけ抽出して簡潔にまとめる
- **表現修整**: 文章をより専門的かつ魅力的なトーンにブラッシュアップ
- **構造化**: 内容に応じて見出しや箇条書きなどのマークダウンを適用
- **深掘り調査**: 選択したテキストに対して追加のプロンプト（方針）を指示し、詳細な調査レポートを生成

### 💾 堅牢なデータ保存と履歴管理
- **リアルタイム自動保存**: 入力が1秒止まるたびにローカルデータベースへ即座に保存。さらに5分ごとの定期バックアップも実行されます。
- **バージョン履歴機能**: 手動保存、自動保存、および「AI処理を実行する直前の状態」を履歴として自動記録。いつでも過去のバージョンを確認して復元可能です。

### 📑 便利なドキュメント管理UI
- **動的目次（TOC）**: 本文中にMarkdownの見出し（`#` や `##`）を入力すると、サイドバーの「目次タブ」に自動生成され、クリックで該当箇所へスクロールできます。
- **調査ノート管理**: AIによる「深掘り調査」の結果は別タブにストックされ、ワンクリックで本文の任意の位置にMarkdown形式で挿入できます。

## 使用技術・ライブラリ

本アプリケーションは、フロントエンド技術を中心に構築されており、外部ライブラリはCDN経由で読み込んでいます。

- **フロントエンド**
  - HTML5 / CSS3 / Vanilla JavaScript
  - **[Tailwind CSS](https://tailwindcss.com/)**: UI全体のスタイリング（CDN）
  - **[FontAwesome](https://fontawesome.com/)**: アイコン描画（CDN）
- **データ管理・パーサー**
  - **[localForage](https://localforage.github.io/localForage/)**: IndexedDBをラップした非同期ローカルストレージ管理
  - **[Marked.js](https://marked.js.org/)**: 調査結果などのMarkdownパースとHTMLレンダリング
- **バックエンド / API**
  - **Cloudflare Workers**: AI処理を仲介するバックエンドAPI（コード内でエンドポイントURLとして指定）

## セットアップ・使い方

### セットアップ
本アプリケーションは単一のHTMLファイルで完結しているため、特別なビルドツールやサーバー環境の構築は不要です。

1. 提供されたHTMLコードを `index.html` という名前で保存します。
2. 保存した `index.html` ファイルを、Google Chrome、Safari、Edgeなどのモダンブラウザで開くだけで直ちに利用を開始できます。
*(※ AI機能との通信やCDNライブラリの読み込みのため、インターネット接続が必要です)*

### 使い方

1. **執筆と保存**
   - エディタエリアに自由にテキストを入力してください。入力内容は自動的にブラウザ内に保存されます。
   - 右上の「手動保存ボタン（フロッピーディスクのアイコン）」で任意のタイミングで履歴を残すことも可能です。
2. **AI機能の利用**
   - エディタ内のテキストをドラッグ（または長押し）して選択状態にします。
   - 画面下部（モバイル）または画面内（PC）に「AIアシスタント」メニューが出現します。
   - 「要約」や「きれい化」などのボタンをクリックすると、AIが処理を実行し、選択したテキストが自動的に書き換わります。
3. **深掘り調査とノートの挿入**
   - テキストを選択して「深掘り調査」を選ぶと、調査方針を入力するダイアログが開きます。
   - 調査を実行すると、サイドバーの「調査タブ」に結果がノートとして保存されます。
   - ノート内の「本文に挿入」ボタンを押すことで、エディタのカーソル位置に調査結果を追記できます。
4. **バージョン履歴からの復元**
   - AIの編集結果が意図したものでなかった場合や、誤って文章を消してしまった場合は、サイドバーの「履歴タブ」を開きます。
   - 過去のバージョン一覧が表示されるので、「このバージョンを復元」をクリックすることで、以前の状態に安全にロールバックできます。
