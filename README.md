# app-aiworkpad

## 概要・説明

**AIWorkPad** は、ブラウザ上で動作するAIアシスタント統合型の高性能なMarkdown対応テキストエディタです。
思考を妨げないクリーンなUIを持ちながら、テキストを選択するだけで呼び出せる「AIフローティングメニュー」や、ドキュメントの文脈を理解する「AIチャット」など、執筆を強力にサポートする機能を搭載しています。

データはブラウザのローカルストレージ（IndexedDB）に自動保存される「ローカルファースト」な設計となっており、サーバーへのログインや煩わしい設定なしで、開いてすぐに安全かつ快適に執筆を始められます。PCだけでなく、スマートフォンなどのモバイル端末でも使いやすいレスポンシブデザインを採用しています。

## 主な機能一覧

### 📝 エディタ＆ドキュメント管理
* **Markdown対応エディタ**: シンプルで書きやすいテキストエディタ。
* **マルチドキュメント管理**: 複数のドキュメントを作成・切り替え可能。タイトルは本文から自動で抽出されます。
* **自動保存＆バージョン履歴**: 執筆中のテキストを自動で保存。AI処理前や手動保存時などにバージョンが記録され、いつでも過去の状態に復元できます。
* **自動目次生成**: 文中の見出し（`#`）を自動解析して目次を生成。クリックで該当箇所へスムーズにスクロールします。

### 🤖 AIアシスト機能
* **AIアクションシート（テキスト選択メニュー）**:
  テキストを選択するだけでポップアップメニューが表示され、以下のワンタップAI処理を実行できます。
  * **きれい化**: 誤字脱字の修正、冗長表現のカット。
  * **要約**: 長い文章を簡潔に要約。
  * **表現修整**: より専門的で魅力的な文章にブラッシュアップ。
  * **構造化**: 箇条書きや見出しを用いて文章を整理・構造化。
* **深掘り調査**: 選択したテキストに対して追加の指示（プロンプト）を与え、AIに詳細な調査・解説を行わせます。結果は「調査ノート」に保存されます。

### 💬 コンテキスト認識AIチャット
* **右サイドパネルのAIアシスタント**: 執筆しながらAIと対話が可能。
* **文脈の共有**: 「ドキュメント全体」や「選択中のテキスト」を含めて質問できるため、文脈に沿った的確な回答が得られます。
* **ワンクリック挿入**: AIの回答や調査結果を、ボタン一つでエディタの本文に挿入できます。

## 使用技術・ライブラリ

本プロジェクトは、モダンなフロントエンド技術を活用し、ビルドツール不要のシングルHTMLファイルとして構築されています。

* **HTML5 / CSS3 / Vanilla JavaScript**: コアロジックとUI
* **Tailwind CSS (CDN)**: 柔軟でモダンなスタイリングとレスポンシブデザイン
* **FontAwesome (CDN)**: 直感的なUIアイコン
* **LocalForage (CDN)**: クライアントサイドでの大容量データ永続化（IndexedDBラッパー）
* **Marked.js (CDN)**: AIチャットや調査ノートのMarkdownパースとHTMLレンダリング
* **Cloudflare Workers (外部API)**: バックエンドのAI処理エンドポイントとして利用

## セットアップ・使い方

### セットアップ
このアプリケーションは単一のHTMLファイル（シングルページアプリケーション）として動作します。面倒な環境構築やパッケージのインストールは不要です。

1. プロジェクトのHTMLファイルをダウンロード、または任意のローカルディレクトリに配置します。
2. HTMLファイルをそのままモダンブラウザ（Chrome, Edge, Safari, Firefoxなど）で開くだけで利用を開始できます。
   *(※より快適に動作させるため、VS CodeのLive Serverなどを用いたローカルサーバー経由での起動を推奨します)*

### 基本的な使い方
1. **執筆を始める**:
   画面中央のエディタエリアにテキストを入力します。Markdown記法が利用可能です。データは自動的にブラウザ内に保存されます。
2. **AIメニューを使う**:
   エディタ上のテキストをマウス（または指）で選択すると、AIメニューがポップアップします。「要約」や「きれい化」などのボタンを押すと、選択箇所がAIの生成結果に直接置き換わります。
3. **ドキュメントを管理する**:
   左上の「フォルダアイコン」をクリックすると、ドキュメント一覧パネルが開きます。「＋」ボタンで新しいドキュメントを作成できます。
4. **ツールパネルを活用する**:
   右上の「ツールアイコン（PCでは常時表示）」から、サイドパネルを開きます。ここでは「AIチャット」「目次」「調査ノート」「履歴」の各タブを切り替えて、執筆の補助として活用できます。
