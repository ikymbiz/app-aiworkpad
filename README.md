# app-aiworkpad

## 概要・説明
**AIWorkPad**は、Google Gemini APIを活用して文章作成を強力にサポートする、ブラウザ完結型のAI搭載Markdownエディタです。

サーバーやデータベースの構築は一切不要で、HTMLファイルをブラウザで開くだけですぐに使用できます。入力したデータやAPIキーはすべてブラウザのローカル環境（IndexedDB / LocalStorage）に安全に保存されます。
テキストを選択するだけで表示される直感的な「AIメニュー」による文章のブラッシュアップや、ドキュメントの内容をコンテキストとして読み込める「AIチャット」、自動保存やバージョン履歴など、執筆活動を効率化するための機能が豊富に詰め込まれています。PCとスマートフォン両方に対応したレスポンシブデザインを採用しています。

## 主な機能一覧

### 📝 Markdownエディタ機能
- リアルタイム入力とプレビュー（WYSIWYGライク）の切り替え。
- 見出し、太字、斜体、引用、コード、リンクなどの入力支援ツールバー。
- リスト（箇条書き・番号付きリスト）のEnterキーによる高度な自動連番機能（ローマ数字やアルファベットにも対応）。
- 見出し(`H1`〜`H6`)から自動生成される目次機能（クリックで該当箇所へスクロール）。

### 🤖 AIテキストアシスト（選択テキスト連携）
テキストをドラッグ選択すると、モダンなフローティングメニューが表示され、以下のAI処理をワンクリックで実行できます。
- **きれい化**: 誤字脱字の修正や冗長な表現の整理。
- **要約**: 選択した長文を簡潔なポイントに要約。
- **表現修整**: より専門的で魅力的な表現へのブラッシュアップ。
- **構造化**: マークダウンを用いた見やすい箇条書きや構造化。

### 🔍 深掘り調査とノート機能
- 選択したテキストを元に、追加のプロンプト（調査方針）を指定してAIに詳細な調査レポートを作成させることができます。
- 調査結果は「調査ノート」として個別に保存され、必要な部分をいつでもエディタ本文に挿入可能です。

### 💬 コンテキスト対応AIチャット
- 執筆中のドキュメント全体、または選択中のテキストを「文脈」としてAIに送信できるチャット機能。
- チャットでのAIの回答をワンクリックで本文に挿入できます。

### 🗂️ ドキュメント・履歴管理
- **複数ドキュメント管理**: 左サイドバーから新しいドキュメントの作成、切り替え、削除が可能です。
- **自動保存**: タイピング後や一定時間ごとにバックグラウンドで自動保存されます。
- **バージョン履歴**: 手動保存、定期自動保存に加え、「AI処理を実行する直前」の状態も自動でスナップショットとして保存されます。いつでも過去のバージョンに復元可能です。

## 使用技術・ライブラリ
このアプリケーションは単一のHTMLファイル（Vanilla JS）で構築されており、以下の技術・CDNライブラリを使用しています。

- **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla JS)
- **スタイリング**: Tailwind CSS (CDN), Tailwind Typography Plugin
- **アイコン**: FontAwesome 6.4.0 (CDN)
- **Markdown解析**: Marked.js (CDN)
- **ローカルデータベース**: LocalForage 1.10.0 (IndexedDBラッパー)
- **生成AI API**: Google Gemini API (`gemini-2.5-flash` モデル)

## セットアップ・使い方

### セットアップ手順
1. このHTMLファイルをダウンロードし、任意の名前（例: `index.html`）で保存します。
2. 保存したファイルをGoogle Chrome, Edge, Safariなどのモダンブラウザで開きます。
3. 画面右上の **設定ボタン（歯車アイコン）** をクリックします。
4. 有効な **Google Gemini APIキー** を入力し、「保存」をクリックします。
   *(※APIキーはブラウザのローカルストレージにのみ保存され、外部サーバーには送信されません)*

### 基本的な使い方
- **文章を書く**: メインエリアで文章を入力します。上部のツールバーを使ってMarkdownの装飾が可能です。
- **AIメニューを使う**: エディタ上のテキストをマウス（またはタップ）で選択すると、AIメニューがポップアップします。目的のボタンを押すと、AIがテキストを処理し、自動で本文を置き換えます。
- **パネルの切り替え**: 画面右上のツールボタン（モバイル）または右サイドバー（PC）から、「チャット」「目次」「調査」「履歴」の各タブを切り替えて使用します。
- **プレビューモード**: ツールバー右側の「プレビュー」ボタンを押すと、Markdownがレンダリングされた読みやすい状態を確認できます。
