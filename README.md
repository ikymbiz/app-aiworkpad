# AIWorkPad

## 概要

**AIWorkPad**は、Googleの「Gemini API」を統合した、ブラウザ上で動作する次世代のMarkdownテキストエディタです。
テキストの執筆から、AIによる自動推敲（きれい化・要約・表現修整・構造化）、チャットベースの壁打ち、深掘り調査まで、あらゆる執筆作業を1つの画面で完結できるように設計されています。

データはすべてブラウザのローカルデータベース（IndexedDB）に自動保存されるため、会員登録や外部サーバーへのデータ保存は不要です。また、PWA（Progressive Web App）に対応しており、スマートフォンやタブレットのホーム画面に追加することで、ネイティブアプリのような感覚で利用できます。

## 主な機能一覧

### 📝 リッチなMarkdownエディタ
- **入力支援ツールバー**: 見出し、太字、斜体、リスト（箇条書き・番号付き・ローマ数字対応）、引用、コードブロックなどをワンクリックで挿入可能。
- **スマート改行**: リスト入力時の自動継続・自動連番機能。
- **プレビュー機能**: MarkdownをシームレスにレンダリングするWYSIWYGライクなプレビューモード。
- **目次自動生成**: ドキュメント内の見出し（`#`）を自動的に抽出し、クリックで該当箇所へスクロールできる目次（TOC）を生成。

### 🤖 AI執筆アシスト（テキスト選択メニュー）
テキストを選択するとフローティングメニューが表示され、以下のAI処理を瞬時に実行できます。
- **きれい化**: 誤字脱字や冗長な表現を自動修正。
- **要約**: 長い文章の要点を簡潔にまとめる。
- **表現修整**: より専門的で魅力的な文章へブラッシュアップ。
- **構造化**: プレーンテキストを適切なMarkdown（見出しやリストなど）に変換。
- **深掘調査**: 選択したテキストに対して、追加の指示や方針を与えてAIに詳細な調査レポートを作成させる。

### 💬 AIアシスタントパネル（サイドバー機能）
- **コンテキスト連動チャット**: ドキュメント全体、または選択中のテキストをコンテキスト（背景情報）としてAIに送信し、精度の高い回答を引き出せます。
- **回答の直接挿入**: チャットの回答や調査結果を、ワンクリックでエディタの本文に挿入可能。
- **調査ノート**: 「深掘調査」で生成されたレポートを専用タブにストックし、いつでも閲覧・引用が可能。

### 💾 安心のデータ管理
- **完全ローカル保存**: すべてのドキュメント、チャット履歴、調査ノートはブラウザ（IndexedDB）に保存されます。
- **自動保存**: タイピング時や一定間隔（5分）でバックグラウンド保存を実行。
- **バージョン履歴**: 手動保存、自動保存、AI処理の直前に自動でバージョン（スナップショット）を作成。いつでも過去の状態に復元できます。

## 使用技術・ライブラリ

本プロジェクトは、ビルド環境不要のシンプルな構成（HTML/CSS/JS）で構築されています。

- **フロントエンド**: HTML5, Vanilla JavaScript
- **スタイリング**: Tailwind CSS (CDN), FontAwesome
- **Markdownパーサー**: Marked.js
- **ローカルデータベース**: localForage (IndexedDBラッパー)
- **AI連携**: Google Gemini API (`gemini-2.5-flash` モデル)

## セットアップ・使い方

### 1. アプリの起動
本アプリは1つのHTMLファイルで構成されています。
ダウンロードしたHTMLファイルをGoogle Chrome, Safari, Edgeなどのモダンブラウザで直接開くか、任意のWebサーバーに配置してアクセスしてください。
（※モバイル端末の場合は、ブラウザのメニューから「ホーム画面に追加」を選択することでPWAとして利用可能です。）

### 2. Gemini APIキーの設定
AI機能を利用するために、GoogleのGemini APIキーが必要です。
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、GoogleアカウントでログインしてAPIキーを作成・取得します。
2. AIWorkPadの画面右上にある **歯車アイコン（設定）** をクリックします。
3. 取得したAPIキー（`AIzaSy...`から始まる文字列）を入力し、「保存」をクリックします。
   *※APIキーはブラウザ内に安全にローカル保存され、直接GoogleのAPIエンドポイントへの通信にのみ使用されます。*

### 3. ドキュメントの作成と編集
1. 左上の **フォルダアイコン** をクリックしてドキュメント一覧を開き、「+」ボタンで新しいドキュメントを作成します。
2. エディタ部分にテキストを入力します。タイトルを入力すると自動的に保存が開始されます。
3. 文章の一部をドラッグして選択するとAIメニューがポップアップします。用途に合わせて「きれい化」などをクリックしてください。
4. 画面右上の **スパナアイコン**（PCでは常に表示されている右サイドパネル）から、AIチャットや目次、バージョン履歴にアクセスできます。
