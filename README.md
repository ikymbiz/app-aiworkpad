# 📝 AIWorkPad

AIをシームレスに統合した、ローカルファーストの高性能Markdownドキュメントエディタです。
テキストを選択するだけでAIによる推敲や要約ができ、ドキュメントの階層管理やチャットアシスタント、記事の自動生成など、ライティングを強力にサポートする機能がすべてブラウザ上で完結します。

## 📖 概要・説明

**AIWorkPad**は、執筆作業とAIアシスタントを一つのUIに統合したモダンなメモ・エディタアプリです。PC・モバイルの双方に最適化されたレスポンシブデザインを採用し、PWAとしてホーム画面に追加することも可能です。

データはすべてブラウザのIndexedDB（ローカルストレージ）に保存されるため、外部データベースの構築は不要。ご自身のGoogle GeminiまたはAnthropic ClaudeのAPIキーを設定するだけで、強力なAIアシスタント機能をすぐに利用できます。

### 主な特徴
- **ローカルファースト**: 全データはブラウザ内に自動保存。オフライン環境でも閲覧・編集可能（※AI機能は要オンライン）。
- **シームレスなAI連携**: エディタ上のテキストを選択すると専用メニューがポップアップし、要約やクリーンアップを瞬時に実行。
- **マルチAIモデル対応**: GeminiとClaudeの両モデルに対応。用途に合わせて切り替え可能。
- **リッチなサイドバー**: ドキュメントのツリー管理（ドラッグ＆ドロップ対応）、AIチャット、記事ドラフト生成、目次、履歴管理をスマートに配置。

## ✨ 主な機能一覧

### 📂 ドキュメント管理
- **階層型ツリー表示**: 左サイドバーでフォルダ・ファイルを無制限（最大10階層）に作成・管理。
- **ドラッグ＆ドロップ**: ドキュメントの並び替えや階層移動が直感的に可能。

### ✍️ 高機能エディタ
- **Markdown入力支援**: ツールバーからワンクリックで見出し、リスト、コードブロックなどの装飾を挿入。
- **リアルタイムプレビュー**: Markdownのプレビューモード切り替え機能。
- **自動保存（オートセーブ）**: 定期的な自動保存により、書きかけのデータ消失を防止。
- **目次自動生成**: ドキュメント内の見出し（H1〜H6）から右パネルに目次を自動生成し、クリックでジャンプ。

### 🤖 AIアシスト機能
- **コンテキストメニュー**: テキスト選択時に表示されるアクションシート。
  - `詳細化` `きれい化（校正）` `要約` `表現修整` `構造化` `深掘調査`
- **コンテキストチャット**: 右パネルのチャットタブで、ドキュメント全体や選択範囲をコンテキスト（前提知識）として含めたままAIと対話可能。
- **記事生成（Draft）**: プロンプト（指示）と参照範囲（ゼロから/ドキュメント全体/選択範囲）を指定して、新規記事のドラフトを自動生成し、エディタへ挿入。
- **AIペルソナ管理**: 「汎用アシスタント」「記事ライター」など、AIの役割（システムプロンプト）を自由に作成・切り替え可能。

### ⏱ バージョン履歴（バックアップ）
- 自動保存、手動保存、AIによるテキスト変更のタイミングでバージョンを自動記録。
- 過去の状態を確認し、ワンクリックで復元可能。

## 🛠 使用技術・ライブラリ

特別なビルドツールは必要なく、1つのHTMLファイルにすべての機能がフロントエンド技術で実装されています。

- **コア言語**: HTML5, JavaScript (Vanilla JS), CSS3
- **スタイリング**: [Tailwind CSS](https://tailwindcss.com/) (CDN / Typography Plugin)
- **アイコン**: [FontAwesome 6.4.0](https://fontawesome.com/)
- **Markdownパーサー**: [Marked.js](https://marked.js.org/)
- **ローカルデータベース**: [LocalForage](https://localforage.github.io/localForage/) (IndexedDBラッパー)
- **バックグラウンド処理**: Web Worker (AI API通信時のUIフリーズ防止)
- **連携AI API**:
  - Google Gemini API (`gemini-2.5-flash`)
  - Anthropic Claude API (`claude-3-haiku-20240307`)

## 🚀 セットアップ・使い方

### セットアップ
本アプリケーションは完全なフロントエンドアプリです。サーバーの構築は必要ありません。

1. 本プロジェクトのHTMLファイル（例: `index.html`）をダウンロードします。
2. ダウンロードしたファイルをブラウザ（Chrome, Edge, Safari推奨）で開きます。
   - または、GitHub Pages、Vercel、Netlifyなどの静的ホスティングサービスにデプロイしてアクセスすることも可能です。
3. 初回起動時、画面右上の **「設定（歯車アイコン）」** をクリックします。
4. ご自身の **Gemini APIキー**（または Claude APIキー）を入力し、「保存」をクリックします。
   - ※ APIキーはブラウザのローカルストレージ内にのみ保存され、外部サーバーに送信されることはありません（各AIの公式APIエンドポイントへの直接通信のみ）。

### 基本的な使い方

1. **ドキュメントの作成**
   - 画面左上の「フォルダアイコン」をクリックして左サイドバーを開き、「+（新規作成）」ボタンからドキュメントを作成します。
2. **AIに文章をブラッシュアップさせる**
   - エディタ上で書いた文章をマウス（またはタップ）で選択します。
   - 「AIメニュー」がポップアップするので、「きれい化」や「要約」ボタンをクリックします。
   - AIが処理を行い、選択した文章が自動的に推敲された内容に置き換わります。
3. **AIチャットを使う**
   - 右上の「ツールパネル（スパナアイコン）」をクリックし、右サイドパネルを開きます。
   - 「チャット」タブを選択し、質問を入力して送信します。
   - 「ドキュメント全体を含める」にチェックを入れると、今書いている内容を踏まえた回答をしてくれます。
4. **記事のドラフトを生成する**
   - 右パネルの「生成」タブを開き、ターゲットやトーン（例: 「初心者向けに分かりやすく」）を入力します。
   - 「記事を生成」ボタンを押すとプレビューが表示され、「エディタに挿入」ボタンで本文に反映できます。
