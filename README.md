# AIWorkPad

## 概要・説明
**AIWorkPad** は、Google Gemini APIを活用してテキスト作成や情報整理を強力にサポートする、ブラウザ完結型のAI統合Markdownエディタです。

サーバーや面倒な環境構築は一切不要。単一のHTMLファイルとして構築されており、データはすべてブラウザのローカルデータベース（IndexedDB）に保存される「ローカルファースト」な設計となっています。

直感的なMarkdown編集に加え、テキストを選択するだけで呼び出せる「AIポップアップメニュー」、ドキュメントの内容を理解した上で回答してくれる「AIチャット」、自動保存やバージョン履歴機能など、執筆活動を加速させる様々な機能が統合されています。モバイル表示にも最適化されており、PWA（Progressive Web App）としてホーム画面に追加して利用することも可能です。

## 主な機能一覧

- **リッチなMarkdownエディタ ＆ プレビュー**
  - ツールバーによる入力支援（太字、斜体、引用、コードブロックなど）とリアルタイムプレビュー
  - 箇条書き・番号付きリストのインテリジェントな自動連番機能
  - 見出し（`#`）から自動生成される目次（TOC）機能とクリックによるジャンプ
- **直感的なポップアップAIアクション**
  - エディタ上のテキストを選択すると、AIメニューが自動でポップアップ
  - **きれい化**: 誤字脱字の修正や冗長な表現の校正
  - **要約**: 長い文章を簡潔に要約
  - **表現修整**: より専門的で魅力的な文章へブラッシュアップ
  - **構造化**: ベタ打ちのテキストを適切なMarkdown（見出し・リスト）に整理
- **深掘り調査ノート**
  - 選択したテキストに対し、追加の指示（調査方針）を与えてAIに詳細な調査レポートを作成させる機能
  - 生成されたレポートは独立した「調査ノート」として保存され、いつでも本文に挿入可能
- **コンテキスト連動AIチャット**
  - 「ドキュメント全体」や「選択中のテキスト」をコンテキストとして含めてAIに質問や相談が可能
  - AIの回答をワンクリックで本文へ挿入
- **堅牢なローカルドキュメント管理**
  - 複数ドキュメントの作成・切替・削除
  - 入力時および5分間隔での「自動保存」機能
  - 最大5世代の「バージョン履歴」を自動バックアップ。AI処理の実行前にも自動保存され、いつでも過去の状態へ復元可能
- **データポータビリティ ＆ その他機能**
  - 全データ（ドキュメント、チャット、調査ノート、履歴）をJSONファイルとしてエクスポート / インポートする機能
  - Web Workerによる非同期AI処理で、処理中もUIがフリーズしない快適な操作性
  - Screen Wake Lock API対応による、長時間のAI処理中の画面スリープ防止
  - レスポンシブデザインによるスマートフォン・タブレットへの完全対応

## 使用技術・ライブラリ

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript (単一ファイル構成)
- **スタイリング**: Tailwind CSS (CDN経由) + Typography Plugin
- **アイコン**: FontAwesome
- **Markdown解析**: Marked.js
- **ローカルデータベース**: LocalForage (IndexedDBの非同期ラッパー)
- **AI連携**: Google Gemini API (`gemini-2.5-flash` モデル)
- **ブラウザAPI**: Web Workers API, Screen Wake Lock API

## セットアップ・使い方

### 1. 起動方法
本アプリはサーバーを必要とせず、ブラウザのみで動作します。以下のいずれかの方法で起動してください。
- ダウンロードしたHTMLファイル（`index.html` 等）を、そのままGoogle ChromeやSafariなどのモダンブラウザで開く。
- 任意のローカルサーバー（VSCodeのLive Serverなど）やWebホスティングサービス（GitHub Pagesなど）に配置してアクセスする。

### 2. 初期設定（Gemini APIキーの登録）
AI機能を利用するには、ご自身のGoogle Gemini APIキーをセットアップする必要があります。
1. 画面右上の **歯車アイコン（設定）** をクリックします。
2. 取得した **Gemini APIキー**（`AIzaSy...`から始まる文字列）を入力し、「保存」をクリックします。
   *(※ APIキーはブラウザのセキュアなローカルデータベース内にのみ保存され、開発者や外部のサーバーに送信されることはありません。)*

### 3. 基本的な使い方
- **執筆とMarkdown入力**
  - 画面左上の「フォルダアイコン」からサイドバーを開き、「＋」ボタンで新規ドキュメントを作成します。
  - 中央のエディタでテキストを記述します。改行時のリスト自動継続など、シームレスな入力が可能です。
- **AIによるテキスト処理**
  - エディタ内のテキストをドラッグして選択すると、AIアクションシートが表示されます。
  - 実行したいアクション（要約、きれい化など）をクリックすると、自動的に処理が行われテキストが置き換わります。
- **サイドパネルの活用**
  - 画面右上の「ツールアイコン（スパナとドライバー）」（PCでは右側に常時表示）から右サイドパネルにアクセスします。
  - **チャットタブ**: ドキュメントの内容をもとにAIと対話ができます。
  - **目次タブ**: ドキュメントの全体構造を俯瞰できます。
  - **調査タブ**: 「深掘り調査」機能で生成した情報を管理します。
  - **履歴タブ**: 過去のバージョンを確認し、いつでも復元できます。
- **データのバックアップ**
  - 設定メニュー（歯車アイコン）内の「データバックアップ」から、全データをJSONファイルとしてエクスポート/インポートできます。ブラウザのキャッシュクリア前や、別の端末にデータを移行する際にご利用ください。
